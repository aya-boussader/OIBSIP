# 🌸 Iris Flower Classification – Oasis Infobyte Internship (Task 1)

This is an Iris Flower Classification project built with Python 
and Machine Learning, developed as part of Task 1 of the 
Oasis Infobyte Internship Program.

## 🛠️ Technologies Used
* Language: Python
* Data manipulation: Pandas, NumPy
* Visualization: Matplotlib, Seaborn
* Machine Learning: Scikit-learn

## 🤖 Models Used & Results

| Model | Accuracy |
|-------|----------|
| KNN | 100% |
| Decision Tree | 100% |
| Random Forest | 100% |

## ✨ Features
* 📥 Load and explore the Iris dataset (150 samples, 3 species)
* 🔍 Exploratory Data Analysis (EDA)
* 📈 Data visualization (histograms, pairplot, confusion matrix)
* 🤖 Train and compare 3 ML models
* 🌸 Predict species of new flowers with confidence percentage
* 🔑 Feature importance analysis (Petal Length 44%, Petal Width 42%)

## 🚀 How to Run Locally

1. Clone the repository
git clone https://github.com/aya-boussader/OIBSIP.git
cd OIBSIP/Task1_Iris_Classification

2. Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn

3. Open the notebook
jupyter notebook iris.ipynb
