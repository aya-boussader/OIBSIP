# 📈 Sales Prediction – Oasis Infobyte Internship (Task 5)

This is a Sales Prediction project built with Python 
and Machine Learning, developed as part of Task 5 of the 
Oasis Infobyte Internship Program.

## 🛠️ Technologies Used
* Language: Python
* Data manipulation: Pandas, NumPy
* Visualization: Matplotlib, Seaborn
* Machine Learning: Scikit-learn

## 🤖 Models Used & Results

| Model | R² Score | MAE | RMSE |
|-------|----------|-----|------|
| Linear Regression | 89.94% | 1.46 | 1.78 |
| Random Forest | 98.13% | 0.62 | 0.77 |
| **Gradient Boosting** | **98.31%** | **0.62** | **0.73** |

## ✨ Features
* 📥 Load and explore the Advertising dataset (200 entries)
* 🔍 Exploratory Data Analysis (EDA)
* 📈 Data visualization (distribution, scatter plots, heatmap)
* 🤖 Train and compare 3 ML models
* 🎯 Predict sales based on advertising budget
* 🔑 Feature importance (TV 61.3%, Radio 38.1%, Newspaper 0.7%)

## 💡 Key Insights
* 📺 TV advertising has the highest impact on sales (61.3%)
* 📻 Radio is the second most important channel (38.1%)
* 📰 Newspaper has almost no impact on sales (0.7%)

## 🚀 How to Run Locally

1. Clone the repository
git clone https://github.com/aya-boussader/OIBSIP.git
cd OIBSIP/Task5_Sales_Prediction

2. Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn

3. Open the notebook
jupyter notebook sales.ipynb
