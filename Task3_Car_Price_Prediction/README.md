# 🚗 Car Price Prediction – Oasis Infobyte Internship (Task 3)

This is a Car Price Prediction project built with Python 
and Machine Learning, developed as part of Task 3 of the 
Oasis Infobyte Internship Program.

## 🛠️ Technologies Used
* Language: Python
* Data manipulation: Pandas, NumPy
* Visualization: Matplotlib, Seaborn
* Machine Learning: Scikit-learn

## 🤖 Models Used & Results

| Model | R² Score | MAE | RMSE |
|-------|----------|-----|------|
| Linear Regression | 84.67% | 1.22 | 1.88 |
| Random Forest | 95.93% | 0.64 | 0.97 |
| **Gradient Boosting** | **96.56%** | **0.56** | **0.89** |

## ✨ Features
* 📥 Load and explore the Car dataset (301 cars, 9 features)
* 🔍 Exploratory Data Analysis (EDA)
* 📈 Data visualization (distribution, boxplots, heatmap)
* 🔧 Feature engineering (Car_Age, Label Encoding)
* 🤖 Train and compare 3 ML models
* 🚗 Predict price of any used car
* 🔑 Feature importance (Present_Price 88.4%)

## 🚀 How to Run Locally

1. Clone the repository
git clone https://github.com/aya-boussader/OIBSIP.git
cd OIBSIP/Task3_Car_Price_Prediction

2. Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn

3. Open the notebook
jupyter notebook car_price.ipynb
